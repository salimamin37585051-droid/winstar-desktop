ICON INSTRUCTIONS ================= Replace icon.svg with your actual icons: - icon.png (256x256 PNG) - for Linux and general use - icon.ico (256x256 ICO) - for Windows .exe - icon.icns (Mac format) - for macOS You can convert icon.svg to these formats using: - https://convertio.co - https://cloudconvert.com - Or any image editor For a quick icon, the GitHub Actions build will use the SVG automatically.
